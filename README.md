## OOP-Chess-Project

Гра в шахи на C++ з використанням **SDL3** та **Dear ImGui**,  
з можливістю гри проти зовнішнього шахового двигуна (наприклад, Stockfish).

---

## Мета проєкту

- Реалізувати повноцінні правила шахів:
  - рокіровка
  - ен-пассан
  - промоція пішака
  - шах / мат / пат
- Зробити приємний 2D-інтерфейс:
  - рендер дошки та фігур через SDL3
  - панелі налаштувань через ImGui
- Додати можливість гри:
  - людина vs людина (локально)
  - людина vs двигун (Stockfish)

---

## Стек технологій

- **Мова:** C++23
- **Графіка:** [SDL3](https://libsdl.org/)
- **UI / debug overlay:** [Dear ImGui](https://github.com/ocornut/imgui)
- **Шаховий двигун:** [Stockfish](https://stockfishchess.org/)
- **Білд-система:** Meson
- **ОС (основна):** Ubuntu Linux/Windows

---

## Можливості

**Поточні:**
- Шахова дошка 8×8, класичний стартовий розклад
- Локальна гра **людина vs людина**
- Кольори сторін: білими / чорними можна обрати при старті

**Заплановані:**
- Візуальне підсвічування:
  - останнього ходу
  - доступних ходів виділеної фігури
- Базовий GUI через ImGui (панелі:
  - налаштувань
  - логів
  - інформації про партію)
- Інтеграція UCI-двигуна:
  - запуск зовнішнього процесу (Stockfish)
  - обмін ходами через UCI-протокол
  - можливість грати **людина vs двигун**
- Збереження / завантаження партій (FEN / PGN)
- Режим аналізу (двигун показує кращі ходи)
- Налаштування рівня сили двигуна (depth / nodes / skill level)
